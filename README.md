# Skillup - Attendance Management System ⏱️

## 📌 Overview
Skillup is a comprehensive full-stack web application designed to manage employee attendance efficiently. It streamlines workforce tracking, automates attendance logging, and provides clear insights for management and HR.

*   **Role-Based Access Control (RBAC):** Secure authentication and distinct dashboards for Admins and Employees.
